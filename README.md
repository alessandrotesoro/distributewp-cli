# Usage
<!-- usage -->
```sh-session
$ npm install -g distributewp-cli
$ distribute-wp COMMAND
running command...
$ distribute-wp (-v|--version|version)
distributewp-cli/0.1.0 darwin-x64 node-v10.13.0
$ distribute-wp --help [COMMAND]
USAGE
  $ distribute-wp COMMAND
...
```
